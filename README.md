# 🏦 WealthPath: Simulador de Planejamento Financeiro para Alta Renda

Este projeto foi desenvolvido com o objetivo de servir como uma ferramenta de apoio à decisão para **Consultores Financeiros**, focando na organização e educação financeira personalizada para clientes de alta renda (HNW).

## 🎯 Objetivo de Negócio
No segmento de consultoria financeira, o maior desafio é converter o patrimônio atual em segurança futura. O **WealthPath** utiliza modelos probabilísticos para demonstrar visualmente ao cliente se o seu estilo de vida atual é sustentável a longo prazo, facilitando o fechamento de propostas de alocação de ativos e seguros.

## 🛠️ Stack Técnica & Diferenciais
* **Python & Streamlit:** Desenvolvimento de uma interface interativa para simulações em tempo real durante reuniões com clientes.
* **Simulação de Monte Carlo:** Diferente de planilhas lineares, este modelo executa 1.000 cenários de mercado, considerando a volatilidade histórica de perfis Conservadores, Moderados e Arrojados.
* **Plotly:** Visualização de dados dinâmica para análise de caminhos patrimoniais e zonas de risco.

## 📊 Funcionalidades Chave
1. **Diagnóstico de Sobrevivência Financeira:** Cálculo da probabilidade do capital durar até a expectativa de vida desejada.
2. **Análise de Sensibilidade:** Permite ajustar aportes e gastos na hora, mostrando o impacto imediato na saúde financeira do cliente.
3. **Perfilamento de Risco Integrado:** Ajuste automático de retorno esperado e desvio padrão com base no perfil do investidor (VMB Advisor style).

## 🚀 Como este projeto agrega valor à VMB Invest?
Este simulador não é apenas um código; é uma ferramenta de **venda consultiva**. Ele permite:
* **Acelerar a conversão:** O cliente visualiza o risco de descapitalização de forma gráfica e estatística.
* **Fidelização (LTV):** Cria um roadmap claro de independência financeira, estabelecendo uma relação de confiança de longo prazo entre o consultor e o cliente.
* **Atitude de Dono:** Demonstra proatividade em criar soluções tecnológicas próprias para otimizar o dia a dia comercial.
